# easy-hugo-new-cmd

Template for hugo new

## how to use

Build

```bash
$ go build -o easy-hugo-new
$ chmod easy-hugo-new
$ sudo mv easy-hugo-new /usr/local/bin/
```

Run cmd in hugo directory

```bash
$ easy-hugo-new
title: sample posts
.../hugo-dir/content/posts/20210303-sample-posts.md created
```
